# 科研工具功能清单

## 后端

- ✅ Flask 基础页面路由
  - 首页聊天页：GET /
  - 知识库管理页：GET /knowledge_base

- ✅ 对话流式接口
  - POST /chat_stream_real，SSE 流式输出
  - 支持文件上下文 + 普通文本问答

- ✅ 主 Agent + 子 Agent 最小架构
  - 主 Agent：意图识别、历史摘要、任务分发
  - 子 Agent：论文解析 / 背景解析 / 专业问答
  - 分发后统一由子 Agent 执行模型调用并返回

- ✅ 双模式运行
  - 测试模式：本地模拟数据，不调用外部模型
  - 正常模式：调用 OpenAI 兼容大模型接口

- ✅ 意图识别（基础版）
  - 测试模式：规则识别
  - 正常模式：大模型识别，失败自动回退规则识别

- ✅ 主/子 Agent 上下文隔离（基础版）
  - 主 Agent 保存有限历史
  - 任务分发前做历史精简摘要
  - 子 Agent 无状态，仅接收摘要 + 实时问题

- ✅ 历史记录管理
  - 维护系统提示词 + 有限轮次消息
  - 支持清空历史（POST /clear_history）

- ✅ PDF 上传与解析
  - POST /upload_pdf
  - 文件大小校验（20MB）
  - 文本提取（PyPDF2）

- ✅ 论文分析结果自动保存
  - 输出保存为 Markdown（含元数据）
  - 保存目录：knowledge/paper/origin_paper

- ✅ 论文知识库自动更新
  - 论文解析场景下自动 create/append
  - 生成分段可视化文件

- ✅ 向量知识库能力（FAISS + 本地向量模型）
  - 论文知识库：创建/追加/查询/统计/inspect
  - 术语知识库：创建/重建/查询/统计/inspect
  - 背景知识库：查询/统计/inspect

- ✅ 术语增强检索（论文子 Agent）
  - 可配置相似度阈值和最大条数
  - 检索结果拼接到子 Agent 提示词

- ✅ 背景增强检索（背景子 Agent）
  - 从背景知识库检索相关上下文后作答

- ✅ 知识库管理 API
  - GET /api/kb/stats
  - GET /api/kb/term/load
  - POST /api/kb/term/rebuild
  - GET /api/kb/<kb_type>/inspect

- ✅ 本地测试数据
  - 新增 utli/agent_test_data.py
  - 分别提供三类子 Agent 的模拟回复模板

- 🔲 主 Agent 历史长度 N 参数化
  - 当前为固定值（代码常量）
  - 计划改为前端可配置并后端动态生效

- 🔲 历史摘要升级为大模型总结
  - 当前是规则截断压缩
  - 计划加入专门的摘要提示词与模型调用

- 🔲 分发与子 Agent 可观测性日志
  - 计划补充意图结果、耗时、tokens、回退原因日志

- 🔲 统一子 Agent 输入输出协议
  - 计划固定 request/response schema，便于后续扩展工具调用

## 前端

- ✅ 聊天基础交互
  - 输入发送、回车发送、字符计数、打字动画
  - Markdown 渲染（marked + DOMPurify）
  - 示例问题快捷发送

- ✅ 文件上传交互
  - PDF 上传入口、状态反馈、错误提示
  - 上传后自动填充提问模板

- ✅ 设置面板
  - API Base URL / API Key / 模型 / 温度
  - 术语检索参数（阈值、最大数量）
  - 本地持久化（localStorage）

- ✅ 测试模式开关
  - 设置页可切换测试模式
  - 请求时把 testMode 一并传给后端

- ✅ 首页模式状态显示
  - 左上角显示当前模式（正常模式/测试模式）

- ✅ 主题切换
  - 日间 / 夜间 / 跟随系统

- ✅ 清空历史按钮
  - 调用后端清空并重置欢迎区

- ✅ 知识库管理页
  - 展示三类知识库统计
  - 查看分段详情
  - 导出论文/背景统计
  - 术语加载、编辑、重建索引

- 🔲 主 Agent 路由可视化增强
  - 计划显示更细粒度分发过程（识别依据、回退说明）

- 🔲 子 Agent 输入摘要可视化
  - 计划提供折叠面板查看“本轮发送摘要”

- 🔲 主 Agent 历史长度 N 前端配置
  - 计划在设置面板增加 N 参数并传后端

- 🔲 调度失败友好兜底提示
  - 计划统一错误文案，区分识别失败/模型失败/检索失败
